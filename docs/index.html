BOARHIRO

Logic. Speed. Minimalism.

A transformer-based neural AI that trains itself continuously and gets smarter the longer it runs. No API keys. No cloud. No cost.

~ install boarhiro
$ pip install boarhiro
Get started View commands

Local Inference

Runs an inference server right on your machine. Zero latency to the cloud.

Always Training

Background training runs forever and saves checkpoints automatically.

Data Hunter

Continuously fetches real text from the web to feed the trainer.

Private by Default

Everything stays on your computer. No API keys, no telemetry.

Commands

Just run boarhiro — that's it.

Command Description
helpShow all commands
statusShow server & trainer status
seturl <url>Point to a remote server
updateUpgrade to latest version
exitQuit

Train manually

$ trainboarhiro # runs forever in background
$ trainboarhiro --no-hunter # use your own data/input.txt
$ stopboarhiro # stop background training
